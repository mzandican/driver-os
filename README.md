# 🚗 Driver OS

A neutral driver-side orchestration system for gig workers.

## Purpose
Helps drivers compare Uber, Bolt, and other jobs to maximize earnings per hour.

## Features
- Job comparison engine
- Earnings tracker
- Smart scoring system
- Multi-platform support (manual input MVP)

## Tech Stack
- React + Vite
- LocalStorage (MVP)
- Modular domain-driven architecture

## Run Locally
```bash
npm install
npm run dev
